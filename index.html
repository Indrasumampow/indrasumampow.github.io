About Me

I'm Indra Sumampow, a passionate Front-End Web Developer enthusiast and a Fresh Graduate in Telecommunication Engineering from Telkom University. Skilled in crafting engaging and responsive user interfaces, I specialize in combining technical expertise with creativity to develop innovative web experiences. Proficient in HTML, CSS, JavaScript, and familiar with modern frameworks like React.

Beyond web development, I am also an entrepreneur, leading ISKA Corporation Indonesia, an export company specializing in Coconut Fiber, Cocopeat, and White Copra. Additionally, as a Co-Founder of Agriture Global Indonesia, I focus on expanding agricultural and coconut-derived exports through strategic planning and business partnerships.

My Projects

As a requirement for my Bachelor's degree in Telecommunication Engineering, my Final Project was developing a Link Power Budget and Rise Time Budget Computation Website with a Daily Report feature to assist network designers and fiber optic technicians. In its development, I used HTML, CSS, JavaScript, Bootstrap, PHP, and MySQL, and hosted the website using Niagahoster. This website not only serves as a computation tool but also provides a Daily Report feature to facilitate systematic recording and reporting of technicians' work results.

I created the Little Lemon Restaurant website interface as a required project to earn a certification from the HTML and CSS In-Depth course. This project showcases an attractive and functional design, demonstrating my ability to implement HTML and CSS elements with creativity.

I created a prototype website interface for Stray Store, an e-commerce platform focused on clothing. The design is responsive, featuring intuitive navigation, interactive elements, and a friendly color scheme. This project served as practice to enhance my web design and development skills.

My team designed and implemented the "Motion Detection Device as a Security System with Internet of Things for Smart Home" project as part of the Internet of Things course. In this project, I used ESP32, PIR Sensor, Buzzer, and Red & Green LED lights. The device functions as a security system, detecting motion and responding with a buzzer sound and LED indicators. This project reflects my understanding and skills in applying IoT concepts to develop smart home security solutions.

I created a personal website interface as a practice project to refine my web design and development skills. The design includes responsive elements, user-friendly navigation, and an appealing color scheme and layout. Through this project, I have practiced combining aesthetics and functionality to create an engaging experience for website visitors.
